Formulieren

Formulieren zijn erg belangrijk op de website, je moet namelijk een eigen Mini aan kunnen maken. Teveneens ook een account. Zonder formulieren gaat dit niet.

Checkbox:

  • Om een checkbox te kunnen maken, begin je simpelweg gewoon met een input zoals bij alle formulieren.
  • Normaal gesproken zou je dan type="tekst" invoeren. Maar in dit geval type je type="checkbox". Hierdoor kan je vakjes aanvinken zoals op bovenstaande foto.
  • Wil je meerdere checkboxen, maak je simepwelg meer inputs.

Account aanmaken

  • Op bovenstaande foto zie je een formulier, waarmee je een account aan kan maken.
  • Dit is erg belangrijk op de website, hierdoor kan je inloggen en krijg je toegang tot je persoonlijke dagboek.
  • Het maken hiervan is heel simpel, je werkt weer met een input. In dit geval input type="tekst"
  • Deze geef je ook allemaal een aparte naam, dit doe je met een placeholder.

Input text:

  • Dit is een erg simpele invoerveld, je hoeft maar 1x een input met een placeholder te gebruiken.
  • Vervolgens maak je eigen aanpassingen in css naar keuze. Dit om er een style aan te geven.

Don'ts:

  • Gebruik geen andere invoervelden(labels) bij het aanmaken van een account buiten Naam, achternaam, datum en opmerkingen.
